# 智能门锁配件业务技能集合

## 描述
本目录包含智能门锁配件业务相关的技能，包括产品说明书生成、小红书爆款文案策划、财务规则制定等。

## 技能列表

### 1. 产品说明书生成技能
- 文件：`产品说明书生成技能.md`
- 功能：使用Python脚本生成产品说明书，包括产品概述、技术参数、安装说明、使用方法、维护保养、注意事项、故障排除和售后服务等内容。

### 2. 小红书爆款文案策划技能
- 文件：`小红书爆款文案策划.md`
- 功能：使用Python脚本生成符合小红书风格的智能门锁配件文案，包括标题、内容和标签。

### 3. 财务规则制定技能
- 文件：`财务规则制定技能.md`
- 功能：使用Python脚本生成财务规则制度，包括财务职责、费用报销、资金管理、固定资产管理、财务监督等内容。

## 使用方法
1. 阅读技能文件，了解技能的功能和使用方法
2. 按照技能文件中的说明，调用相应的函数
3. 对生成的结果进行适当的修改和调整

## 工具依赖
- Python 3.x
- python-docx（用于生成Word文档）
